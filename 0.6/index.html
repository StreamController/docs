    Installation

    • GitHub:

      This is currently the only way to install StreamController, because it is still in heavy development.

      1. git clone https://github.com/Core447/StreamController
      2. cd StreamController
      3. pip install -r requirements.txt
      4. Finished! Open the app by running python3 main.py
